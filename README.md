# Grit : Git from scratch
## Is an attempt to create a Git like VCS
## Attempted to implement basic functionalities

## Supported commands :
###  `grit init`
###  `grit add <file>`
###  `grit commit <file> <message>`
###  `grit log`
###  `grit show <commit hash>`
